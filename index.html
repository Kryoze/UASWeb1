Soal No.1

Jelaskan cara kerja Website serta jelaskan hubungan antara Web Server, Domain , dan Hosting!

Cara kerja website secara singkat adalah sebagai berikut:

  • Ketika pengguna mengetikkan nama domain di browser, browser akan menghubungi DNS (Domain Name System) untuk mencari alamat IP yang sesuai dengan nama domain tersebut.
  • DNS akan mengembalikan alamat IP yang terkait dengan nama domain ke browser.
  • Browser akan mengirimkan permintaan HTTP (Hypertext Transfer Protocol) ke web server yang memiliki alamat IP tersebut.
  • Web server akan memproses permintaan HTTP dan mengirimkan respon berupa halaman web yang diminta, beserta file-file pendukung seperti gambar, CSS, JavaScript, dll.
  • Browser akan menampilkan halaman web yang diterima dari web server kepada pengguna.
Hubungan antara web server, domain, dan hosting adalah sebagai berikut:
  • Web server adalah tempat di mana hosting berada. Hosting menyediakan ruang penyimpanan untuk website dan data-data lainnya.
  • Domain adalah alamat yang digunakan untuk mengakses website yang disimpan di hosting. Domain memudahkan pengguna untuk mengingat dan menemukan website tanpa harus menghafal alamat IP.
  • Hosting dan domain saling berhubungan melalui DNS, yang bertugas untuk menghubungkan nama domain dengan alamat IP web server.
